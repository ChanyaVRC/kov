# Usage

kov has three subcommands:

| Command | Description |
|---|---|
| [`encode`](encode.md) | Compress a file |
| [`decode`](decode.md) | Decompress a file (full or partial) |
| [`info`](info.md) | Show seek table metadata |

All subcommands accept `--help` for a full option list:

```sh
kov encode --help
kov decode --help
kov info --help
```
